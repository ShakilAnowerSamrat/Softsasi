## üíª Installation & Usage

Since this is a static website, no complex build process is required.

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Start-B-H-M/Softsasi.git
    ```
2.  **Navigate to the project folder**:
    ```bash
    cd Softsasi
    ```
3.  **Run Locally**:
    *   Simply look for the `index.html` file in the root directory.
    *   Double-click to open it in any modern web browser (Chrome, Firefox, Edge, Safari).
    *   Alternatively, you can use a simple static server like VS Code's "Live Server" extension for a better development experience.
